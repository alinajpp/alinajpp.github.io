    Headings

    Headings are used to index the structure and content of web pages. Users skim a page by the headings, so it's important to show what they will be seeing before they see it. Tags can range from <h1> to <h6>, in descending order in size.

    <h1>

    <h2>

    <h3>

    <h4>

    <h5>
    <h6>

    The <hr> tag can be used to break headings, and are the lines seen in between each section on this webpage. The tags used are, respectively, <h1>, which is placed at the beginning of a heading, and </h1>,which is placed at the end of a heading.

    W3Schools Page

    Paragraphs

    Paragraphs are what they are; a plain block of words that a user reads to get into detail. No matter the amount of extreneous spaces and breaks put in a paragraph, the source code will be displayed the same way and those breaks ignored.

    The only way to make a break in a paragraph is with the <br> tag--the HTML equivalent of the enter bar. Tags used are the <p> tag at the beginning and the </p> tag at the end. This block of words was created using the paragraph tags.

    W3Schools Page

    Comments

    Comments are used to place notifications and reminders in HTML without seeing them on the display. The tags used are <!-- at the beginning and --> at the end. There could be comments inserted here and you wouldn't be able to see them.

    W3 Schools Page

    Links

    Links are created simply by inserting the link between the tags <a href=" and ">. Afterwards, the text representing the link is written out, and the tag </a> added at the end. See table above for examples; the "W3Schools Page" below is a link. With a local link, the entire url does not have to be added, only the defined ending. With this Github page, inserting <a href="index.html">Home</a> will take you straight to the homepage.

    Images

    Images are defined with the tag <img src="url">, with the image address inserted in between. In the case that an image will not load, the alt attribute must be applied with the alternate text. The width and height of the image can be customized. Gifs also work with the image attribute, and the gif address must be inserted the same way.

    Tables

    An HTML table is defined with the <table> tag. Each row is defined with <tr> and a header is defined with <th>. The data in each row is preceeded with <td>.
    With the code:

    <table>
    
      <tr>
    
      <th>First Name</th>
    
      <th>Second Name</th>
    
      </tr>
    
      <tr>
    
      <td>Alina</td>
    
      <td>Jitpraphai</td>
    
      </tr>
      <td>Carolyn</td>
    
      <td>Cui</td>
    The following table will be created:

    First Name Second Name
    Alina Jitpraphai
    Carolyn Cui

    To see a better defined table, look at the top of this page.
